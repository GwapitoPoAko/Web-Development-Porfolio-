# Web Development Portfolio

This is the personal portfolio of Nyko Paolo P. Guillero, built to showcase my academic and personal projects as an Information Technology student.

## About the Project
This portfolio serves as a digital space to present:
*   **Projects**: Including the Tunhay Grocery System and EduTor mobile application.
*   **Skills**: My current toolkit including HTML, CSS, JavaScript, PHP, Laravel, and React Native.
*   **Achievements**: Certifications and academic milestones.

## Built With
*   **HTML5**
*   **CSS3** (with custom dark/light theme support)
*   **JavaScript** (for animations, interactions, and lightbox functionality)
*   **Boxicons** (for iconography)

## How to View
You can view the live site or explore the source code directly in this repository. The project uses a custom grid layout for project cards, allowing you to click on project images to view them in a full-screen lightbox.
